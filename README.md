# NextJS_Moralis_installer

This python tool installs a NextJS website with all the moralis integration

## Requirements
Python3

## How to Use
`python3 /PATH/TO/nextjs_installer.py`

tool will ask you what you want to name your project, installs NextJS in {YOURPROJECT}

`cd {YOURPROJECT}`

edit `.env.local` and setup your moralis api information

`npm i && npm run dev`
